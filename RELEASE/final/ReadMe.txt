/!\ Si les boutons afichant l'aide et la maille personnelle nefonctionnent pas : 

C'est que vous vous êtes trompé de version du logiciel ! Elle est incompatible avec vote version de Windows, et
il vous faut alors le désinstaller puis réinstaller la bonne version (32 ou 64 bits).



Rappel : pour voir quelle version de Windows vous possédez :
 
		- Cliquez droit sur l'onglet ORDINATEUR du menu DEMARRER
		- puis cliquez sur PROPRIETES
		- La version de windows sous laquelle vous vous situez est affichée dans le fenêtre que vous venez d'ouvrir !
